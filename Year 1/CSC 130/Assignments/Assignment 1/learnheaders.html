The easy way!

Headers make up headings of text. They're similar to headings represented common word processors. Such they can be relevant to define beginnings of text, titles, etc..
Heading are written in HTML as:
<h> Heading Content </h>
Where the # is of some value from 1 to 6, where a value of 1 has a greater size than a value of 6.
Example would be < h3 > This is a header </h3> would look like

This is a Header


Hence the bigger the value, the larger the header. <
