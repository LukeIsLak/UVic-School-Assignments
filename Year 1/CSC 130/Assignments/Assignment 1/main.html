Learning HTML

The easy way!

Learning the foundation of html is fairly easy compared to other languages. Using easy to learn --although one of a kind-- syntax alongside text based data to define websites. This website is used to elaborate the fundamentals of HTML, although other resources such as CSS and Javascript can be used to make your website to the next level.

HTML is boiled down to elements and the contents within elements.
Elements are used to display content within a page. Each element has a different purpose and are displayed/visualized differently. These use syntanx of < element type > to define the start of an element and < / element type > to define the end of an element. This applies to most elements. An example of this can be seen here:
< Element Type> Content Within the Element< /Element Type>
These make up the majority of your page.

Content within elements are fairly simple. For most of the elements it applies an text to the said element. This is more explained for the each of the elements when learning about them specifically.

Headers make up headings of text. They're similar to headings represented common word processors. Such they can be relevant to define beginnings of text, titles, etc..
Heading are written in HTML as:
<h> Heading Content </h>
Where the # is of some value from 1 to 6, where a value of 1 has a greater size than a value of 6.
Example would be < h3 > This is a header </h3> would look like

This is a Header


Hence the bigger the value, the larger the header.

Paragraphs is a fundamental of inserting text within your website using HTML.
Parapgraphs are written in HTML as:
<p> Paragraph content </p>
Example would be < p > This is a Paragraph, Hello World </p> would look like

This is a Paragraph, Hello World

Lists are a common way to list multiple elements into a single organized element.
There a multiple ways to display lists in. For example there are unordered lists (ul), and ordered lists (ol).
To write the elements inside the lists, you uses a <li>Element Content</li>
Unordered lists simply use bullet points to display your elements.
Unordered lists are written in HTML as (as many elements can be added as wanted):

<ul>
<li> Element 1 <li>
<li> Element 2 <li>
<li> Element 3 <li>
</ul>

This would look like
  • Element 1
  • Element 2
  • Element 3


Ordered lists use numbered points to display your elements.
ORdered lists are written in HTML as (as many elements can be added as wanted):
<ol>
<li> Element 1 <li>
<li> Element 2 <li>
<li> Element 3 <li>
</ol>

This would look like
  1. Element 1
  2. Element 2
  3. Element 3
